APA ITU NUMPY?

Numpy adalah salah satu library python yang digunakan untuk analisis data numerik.data numerik adalah data yang berupa angka.

cara pakai Numpy

#pertama-tama kita impor library numpy terlebih dahulu

import numpy as np

#contoh program sederhana menggunakan numpy

#contoh variabel
A=np.array([1,2,3,4,5])
B=np.array([6,7,8,9,10])
print(A,B)

#contoh operasi penjumlahan

print('penjumlahan')
print(A+B)

#contoh operasi pengurangan

print('pengurangan')
print(A-B)

#contoh operasi perkalian

print('Perkalian')
print(A*B)

Jika kita jalankan program python di atas maka hasilnya akan seperti ini.
